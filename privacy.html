Privacy Policy

Last updated: May 18, 2026

This Privacy Policy describes how Bagmanov AI Publisher ("we", "our", or "the application") collects, uses, and protects information when you use our application.

1. Information We Collect

Bagmanov AI Publisher is a personal content publishing tool. The application accesses Pinterest account data solely for the purpose of creating and managing pins on behalf of the authenticated user. We collect only the minimum data necessary to provide this service:

  • Pinterest account username and public profile information
  • Pinterest boards list (to select where to post)
  • Access tokens provided by Pinterest OAuth

2. How We Use Your Information

We use the collected information exclusively to:

  • Authenticate you with Pinterest via OAuth 2.0
  • Create and publish pins to your Pinterest boards
  • Read your boards list to allow board selection

We do not sell, trade, or share your personal information with third parties.

3. Data Storage

Access tokens are stored locally in your browser (localStorage) and are never transmitted to any external server other than Pinterest's official API endpoints. No personal data is stored on our servers.

4. Pinterest API Usage

This application uses the Pinterest API in accordance with Pinterest's API Terms of Service. We only request the permissions necessary for core functionality: pins:read, pins:write, boards:read, and user_accounts:read.

5. Security

We implement industry-standard security measures to protect your information. All communication with Pinterest's API is conducted over HTTPS.

6. Your Rights

You can revoke this application's access to your Pinterest account at any time by visiting Pinterest Settings → Security → Apps and revoking access to "Bagmanov AI Publisher".

7. Changes to This Policy

We may update this Privacy Policy from time to time. We will notify you of any changes by updating the "Last updated" date at the top of this page.
